# asolide
Simple IDE for demonstration

## Requirements
- Node.js, npm
- bower
- Altera Quartus
  - `quartus_sh`
- nslcore
  - `nsl2vl`
- Terasic DE2-115 FPGA Board

## Usage

```
$ git clone https://github.com/ishikuro/asolide.git
$ npm install
$ bower install
$ cp /path/to/mkproj-DE2_115.tcl workspace/
$ node app.js
```

Open http://localhost:3000/
